최소 신장 트리라고도 불린다.

크루스칼 알고리즘이다.

수정 예정

```python
def find_parent(x):
    if parent[x] != x:
        parent[x] = find_parent(parent[x])
        
    return parent[x]

def union(x, y):
    x = find_parent(x)
    y = find_parent(y)
    
    if x < y:
        parent[y] = x
    else:
        parent[x] = y      


```
